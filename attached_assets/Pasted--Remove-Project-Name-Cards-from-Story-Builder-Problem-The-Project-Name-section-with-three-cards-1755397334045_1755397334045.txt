# Remove Project Name Cards from Story Builder

## Problem
The Project Name section with three cards is appearing in the Story Builder, but it should only be in the Dashboard and other builder pages (World, Production, Asset, Script, Deck).

## Solution
Update the Story Builder layout to remove the Project Name cards section and give the AI Assistant the full bottom area.

### Replace the Story Builder section with this:


## Key Changes:

1. **Removed Project Name section** - No more three project cards in Story Builder
2. **Clean AI Assistant** - AI Assistant gets the full bottom area without project cards above it
3. **Proper proportions** - Story editor gets most of the space, AI gets a clean fixed 200px at bottom
4. **Story-specific layout** - Different from other builder pages which will keep the project cards

## Result:
- ✅ Story Builder: Clean layout with just Story Editor + AI Assistant (no project cards)
- ✅ Other builders (World, Production, Asset, Script, Deck): Keep project cards + AI Assistant  
- ✅ Dashboard: Keeps its unique layout with project cards

The red box section will be completely removed from the Story Builder, giving you a clean interface with just the story editor and AI assistant.
